# QT
QT（C++）开发小众日记app项目
# 环境
1. QT creator 5.5.0
2. win10
3. C++11以上（lambda表达式）
4. mingw（qt creator有的版本自带）
# 测试账号：1 密码：1
# 注意事项
1. 利用qt creator打开Layout.pro进行开发即可。
2. images 文件存储了用到的图片文件。
3. app采用SQLite数据库进行数据管理。
4. 项目基本功能已经实现，仍在开发阶段。
5. 该项目可作为C++练手项目。
